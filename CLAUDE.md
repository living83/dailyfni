# 사용자 작업 규칙

## 결과 전달 방식
- 작업(커밋/푸시)을 마치면 항상 메시지 끝에 **SSH 서버에서 바로 복사·실행 가능한 배포 명령어**를 코드 블록으로 적는다.
- 서버 프로젝트 경로: `/opt/dailyfni`
- 기본 형식:
  ```bash
  cd /opt/dailyfni && git pull origin <작업한 브랜치명> && pm2 restart all
  ```
- 여러 브랜치에 작업했으면 브랜치별로 각각 적는다.
- 스키마/마이그레이션 변경이 포함된 경우 `pm2 restart all` 앞에 `npx prisma migrate deploy` 를 추가한다.

## 옵션 제안
- 선택지를 물을 때는 **최대 2개**만 제시하고, 충분히 검토한 뒤 하나에 "(권장)" 표시를 붙인다.
